## 常见问题速查

<AccordionGroup>
  <Accordion title="如何配置 API 密钥？">
    阅读 [配置与环境](/detailed_guide/config)。重点关注 `provider.json` 的提供商到模型列表结构。
  </Accordion>
  <Accordion title="装饰器支持同步函数吗？">
    不支持。`@llm_function`、`@llm_chat` 和 `@tool` 都要求装饰 `async def` 函数。详见 [llm_function 装饰器](/detailed_guide/llm_function)。
  </Accordion>
  <Accordion title="如何做多轮对话？">
    从 [llm_chat 装饰器](/detailed_guide/llm_chat) 开始，了解 `history`、`stream=True` 和返回模式。
  </Accordion>
  <Accordion title="如何中断当前回复？">
    阅读 [中断与取消](/detailed_guide/abort)，使用 `AbortSignal` 控制运行中回合。
  </Accordion>
  <Accordion title="如何让 LLM 调用函数或外部 API？">
    阅读 [工具系统](/detailed_guide/tool)，学习 `@tool`、返回类型和工具最佳实践注入。
  </Accordion>
  <Accordion title="支持哪些 LLM 提供商？">
    阅读 [LLM 接口层](/detailed_guide/llm_interface)。框架面向 OpenAI Compatible 接口设计，可接入多种兼容服务。
  </Accordion>
  <Accordion title="如何处理错误和重试？">
    先看 [LLM 接口层](/detailed_guide/llm_interface) 的故障排除，再结合日志和事件流排查。
  </Accordion>
</AccordionGroup>
